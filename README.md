# 行政機關辦公日曆及發票兌獎查詢機器人 #
### 這是一個多功能的機器人，她提供以下服務：
1.  查詢補班、補假、調整放假和節日的日期
2.  查詢發票兌獎號碼
3.  有選擇困難嗎?她可以幫你選 
### 依照指示輸入數字即可開始查詢
